Latest Release

Release notes for the newest CoreNuc Poker League Manager build, including the changes most useful to tournament hosts and league organizers.

CoreNuc Poker League Manager 1.2.1.0

Version 1.2.1.0 is the latest packaged release for the Windows app. This release focuses on live tournament control, stronger tournament display feedback, improved Store-ready presentation assets, and quality-of-life updates for running a home poker night.

Version 1.2.1.0
Build 1210
Package Date May 24, 2026
Primary Platform Windows

What Changed

Live Tournament Sounds

Change sounds during active play

Tournament sound choices can now be changed while a tournament is already running, making it easier to tune the room experience without restarting the display.

Round Warnings

One-minute warning popup

The tournament display and timer display now support a one-minute warning popup before a round ends, helping hosts and players prepare for the next blind level.

Game Setup

Bust-out visibility in setup

Bust-out information was added to the game setup flow so tournament directors can review more of the current game state from the setup area.

Tournament Display

Better display behavior

The main display, timer display, and related view models received updates that make live tournament state easier to manage and easier to present.

Store Assets

Updated screenshots and logos

Store-ready screenshots and logo assets were prepared for the release, including views for setup, tournament display, player actions, TV browser display, tables, stats, settlement, and game details.

Support Website

Public support pages

Public website pages were added for product overview, help content, privacy policy, technology details, screenshots, and release information.

Included App Areas

  • Texas Holdem game setup for players, rounds, chips, buy-ins, add-ons, rebuys, prizes, and tables.
  • Live tournament display with round timer, blind levels, antes, recent actions, and tournament controls.
  • Player action tools for rebuys, add-ons, bust outs, receipts, restores, and undo support.
  • TV Browser Display for showing selected tournament screens on another browser on the same home network.
  • Tables and seating tools for assigning seats and balancing active players.
  • Player stats, league standings, settlement reports, and game details review.
  • Settings for display options, receipt content, database import/export, default game data, and tournament sounds.

Packaging Notes

The latest Windows package is prepared as a Microsoft Store style package for CoreNuc Poker League Manager. Store licensing and purchase handling are managed through the Microsoft Store listing.

The packaged artifact for this release is named Holdem.WinUI_1.2.1.0_x64.msix.

Known Notes

  • The desktop app remains the primary tournament display and source of truth.
  • The TV Browser Display should be used only on trusted local networks.
  • Phone remote connection is not enabled in this build.
